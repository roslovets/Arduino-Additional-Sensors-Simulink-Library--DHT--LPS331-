# Arduino Additional Sensors Library

Work with additional sensors

## Supported sensors

- DHT11
- DHT21
- DHT22
- AM2301
- LPS331

## How to install

1. Download and unpack library to any folder **without spaces in the path**
2. Run `INSTALL`
